# Symfony Tag Bundle

`TaggableTrait` for your entities to assign multiple tags per entity, and `TagsType` field for forms.

## Requirements

- PHP 8.1+
- Symfony 6.4+
- Symfony Stimulus
- Symfony UX Turbo & UX Autocomplete
